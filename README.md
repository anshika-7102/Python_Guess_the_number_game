# Guess The Number Game 

A simple Python-based number guessing game where the computer randomly selects a number between 0 and 100, and the player has to guess it correctly.
The game provides hints after every guess:

* If the guessed number is too high, it suggests a smaller number.
* If the guessed number is too low, it suggests a greater number.

## Features

* Random number generation using Python's `random` module
* Interactive user input
* Hint system for better gameplay
* Beginner-friendly Python project

## Technologies Used

* Python

## How to Run

1. Install Python
2. Run the program using:

```bash
python number_guess.py
```
